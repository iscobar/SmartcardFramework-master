Those projects have been originally published on Codeproject. They provide a smartcard framework for .NET that can be used as a starting point in your project if you need a smartcard support using the basic APDU protocol and a card reader that is compliant with PC/SC.

The articles on codeproject can be found at the following addresses:
http://www.codeproject.com/Articles/16653/A-Smart-Card-Framework-for-NET
http://www.codeproject.com/Articles/17013/Smart-Card-Framework-for-NET
